AWS Big Data Certified - Fundamentals

The demand for Cloud Architect and Cloud Big Data Specialist will explode in the next 5-10 years as more and more traditional companies start migrating to the cloud as their data lake. AWS is the leader in cloud technologies. Getting AWS Certified will give a cloud/data professional a big advantage in the job market.


The AWS Big Data Fundamentals course is the first in a series of 3 courses to help you get certified as AWS Big Data Specialist. It helps the students build fundamental knowledge about Big Data on AWS and focus on learning the most essential tools.


This course introduces the core AWS Big Data related services, including S3, EMR, Redshift, Kinesis, Glue, Athena, QuickSight and Machine Learning. Students will learn how to work with various tools via hands-on labs.

LEARNING OUTCOME

Upon completing the course, students will be able to:

  • Gain hands-on experience with the core AWS Big Data tech stacks.

  • Build an architecture for the most common scenarios, including streaming, storage, process, analyzing and visualization, on AWS.

  • Understand the trade-off between technical designs. Understand cost consideration and operational consideration.

  • Target AWS related jobs that pay 80K+ annually

About This Course

Who is this course for?

    This course is designed for students and professionals who want to become AWS Certified. More specifically, it is developed for
  • College students who expect to find big data or data analytics related software development job.
  • Experienced IT engineers (embedded software engineer, test engineer, automation engineer, network engineer, front-end engineer, mobile developer, game developer...) who want to become AWS big data professionals
  • Experienced Big Data engineers who are interested in extending the knowledge to AWS and most cutting-edge Big Data technologies
  • Data related non-software engineer roles (Business Intelligent Engineer, Data Engineer, Data Analyst, Data Scientist...) who want to gain software architect knowledge for career growth.
  • The data enterprises that plan to migrate their legacy system to AWS based architect
